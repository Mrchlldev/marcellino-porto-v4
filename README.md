# Drift Portfolio - Marcellino F.A

Modern portfolio website using Next.js, TypeScript, Tailwind CSS, and AOS Blur animation.

## Run Local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Edit Content

All editable text, links, projects, skills, education, and car lineup settings are in:

- `lib/settings/site.ts`
- `lib/settings/content.ts`

## Deploy to Vercel

Upload the repository to GitHub, then import it from Vercel. No environment variables are required.

## Note

Car brand names are used only as inspiration labels. The design uses custom CSS/SVG-style silhouettes and no official brand assets.
